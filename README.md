🛡 CDR Forensic Platform

A Content Disarm & Reconstruction (CDR) security platform for detecting and neutralizing threats in PDF and OOXML documents.

🚀 Features
📄 PDF Sanitization (JavaScript, links, embedded objects removal)
📊 IOC Extraction (Domains, IPs, URLs)
📁 OOXML Support (PPT, DOCX, XLSX)
🧬 Threat Detection Dashboard
📉 Attack Surface Reduction Graphs
📤 CSV Export & Reports
🧠 Tech Stack
Python
Flask
SQLite
PikePDF
Matplotlib

⚙️ Setup Instructions

1. Clone repo
   git clone https://github.com/Kamalhas2005/cdr-forensic-platform.git
   cd cdr-forensic-platform
2. Install dependencies
   pip install -r requirements.txt
3. Run application
   python app.py

   Web dashboard for analysis
   Sanitized files
   IOC intelligence
   Security reports
